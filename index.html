Installation

Copy all files contained in “source” folder to your web space via FTP (Upload to any directory).

Now login to your phpmyadmin, create a new database and import database.sql to newly created database.

Open configurations.php file located in [Php Password/Pattern Protect \ application] with any text editor

    - Change database configurations like host, database name, database user name and password

    - Replace ['base_url' => 'http://127.0.0.1/phplogin'] with Php Password/Pattern Protect url. Like ['base_url' => 'http://example.com/Php Password/Pattern Protect']         on line 44

    - If you don't want new users to signup then replace ['allow_signup' => true,] with ['allow_signup' => false,] on line 80

    - If you don't want users to change their password ['change_pass' => true,] with ['change_pass' => false,] on line 76

Navigate with your browser to Php Password/Pattern Protect url - [e.g http://example.com/phplogin/login.php]

Migration

If you are moving the script already installed and configured in a new location (from local to server, from server to local, or in a different directory) remember to change <?php require_once("Php Password/Pattern Protect/application/session.php"); ?> according to new location

How to protect pages

Add this single line at the top of the page to protect it with login <?php require_once("application/session.php"); ?>

Note: web page extension must be .php 

<?php require_once("application/session.php"); ?>

and page begins here

Redirecting url's

If the user goes to protected page and is not logged in, then he/she will be redirected to login page and after successfull login he/she will be redirected back to same protected page.

If you want to redirect user to custom URL then login.php?return=YOUR CUSTOM URL if another website then must include http:// or https:// login.php?return=https://fb.com

Change Password

Logged in users can change their passwords by going to http://example.com / script folder / changepass.php

If you don't want to allow users to change pasword then

     Open configuration.php file in any text editor and replace 'change_pass' => true, with 'change_pass' => false, on line 76
